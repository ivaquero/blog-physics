Bivariate Normal Distribution¶

The most prevalent multivariate normal distribution is bivariate normal distribution

\[ \begin{equation} f_\mathbb{X}(x_1,...,x_2)=\frac{1}{\sqrt{(2π)^k|Σ|}}\exp{\Big(-\frac{(x-μ)^TΣ^{-1}(x-μ)}{2}\Big)} \end{equation} \]

There are two ways of formulating a multivariate normal distribution in Python, use any one you see fit.

1st Method of Formulation¶

Random.seed!(42);
mu_x = 0;
sigma_x = 2;
mu_y = 0;
sigma_y = 2;

d_mvnorm = MvNormal([mu_x, mu_y], [sigma_x 0; 0 sigma_y]);

x = LinRange(-10, 10, 100);
y = LinRange(-10, 10, 100);
z = [pdf(d_mvnorm, [i, j]) for i ∈ x, j ∈ y];

fig = Figure(resolution = (1000, 600));
ax = Axis3(
    fig[1, 1],
    xlabel = "X axis",
    ylabel = "Y axis",
    zlabel = "Z axis",
    title = "Bivariate Normal Distribution"
);

surface!(ax, x, y, z);

fig

2nd Method of Formulation¶

Random.seed!(42);
mu_x = 0;
sigma_x = 7;
mu_y = 0;
sigma_y = 15;
d_mvnorm2= MvNormal([mu_x, mu_y], [sigma_x 0; 0 sigma_y]);
z2 = [pdf(d_mvnorm2, [i, j]) for i ∈ x, j ∈ y];

fig = Figure(resolution = (550, 400));
ax = Axis(fig[1, 1])

contourf!(ax, x, y, z2)
fig
